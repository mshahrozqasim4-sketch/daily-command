# Shahroz — Daily Command

Personal discipline tracker with push notifications.

## Deploy to GitHub Pages (5 minutes)

1. Go to github.com → New repository
2. Name it: `daily-command` (or anything you want)
3. Upload all 4 files: `index.html`, `sw.js`, `manifest.json`
4. Go to Settings → Pages → Source → main branch → Save
5. Your app will be live at: `https://YOUR-USERNAME.github.io/daily-command`

## Features
- Live clock and date
- 29-day launch countdown (updates in real time)
- 8 daily alarms with countdowns
- Daily checklist with progress bar
- Discipline streak tracker (30-day history)
- Push notifications (browser permission required)
- Works as a PWA — add to home screen on mobile

## Alarms
| Time | Purpose |
|------|---------|
| 7:00 AM | Wake up — eat breakfast |
| 7:30 AM | Deep work begins |
| 12:30 PM | Lunch + walk |
| 1:30 PM | Second work block |
| 5:30 PM | Work ends |
| 6:30 PM | Light work / planning only |
| 9:00 PM | No screens |
| 10:30 PM | Sleep |

## To enable notifications
Open the app → tap the yellow banner → Allow

## To add to phone home screen
Chrome/Safari → Share button → "Add to Home Screen"
